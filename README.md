🚀 ERP SAP Ariba Automation Platform

A modern AI-powered ERP automation platform designed to simplify and accelerate SAP Ariba data processing workflows.

This platform combines:

* ⚡ Python FastAPI API backend
* 🎨 React frontend
* 🤖 AI-powered sheet detection
* 📊 Dashboard & analytics
* 🔐 JWT + OTP authentication
* 📁 Multi-file processing
* 🧾 SAP Ariba export automation
* 📂 File traceability system

The system works through a powerful API architecture built with Python and FastAPI, allowing multiple frontends (React, Streamlit, or future applications) to interact with the same backend processing engine.

The platform automatically analyzes uploaded files, detects SAP Ariba sheet structures, maps columns intelligently, and generates production-ready Ariba-compatible outputs.
